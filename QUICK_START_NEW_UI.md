## Animation Types

### Entrance
- `fade-in`: Opacity + translateY (0.5s)
- `scale-in`: Scale from 95% to 100% (0.3s)
- `slide-up`: TranslateY from 20px (0.4s)

### Attention
- `pulse-glow`: Pulsing shadow (2s infinite)
- `floating`: Subtle levitation (3s infinite)
- `shimmer`: Loading effect (1.5s infinite)

### Interactive
- `hover`: Transform + shadow growth
- `active`: Scale down slightly
- `focus`: Ring appearance
